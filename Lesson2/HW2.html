Почему код дает именно такие результаты?

var a = 1, b = 1, c, d;

c = ++a; alert(c); // 2

префиксная форма ++a увеличит counter и возвратит новое значение 2.


d = b++; alert(d); // 1

постфиксная форма b++ вернет значение до увеличения


c = (2+ ++a); alert(c); // 5

на момент выполнения a уже равно 2. префиксная форма к 2 прибавит единицу

Тем самым получаем выражение c= 2+3


d = (2+ b++); alert(d); // 4

На момент выполнения b = 2 соответственно постфикс вернет значение до увеличения

то есть выражение будет b = 2+2
